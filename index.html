Logica del Sistema — Manuscrito
01
True Range con Gap
El rango verdadero incluye el gap overnight. Si el mercado gapea, el rango efectivo del dia es mayor. Esto define los niveles PIVOT correctamente.
TR = prev_range + |gap|
PIVOT = prev_high − TR/2
02
1P — Initial Point
El mercado tiende a moverse 25% del TR75 en la direccion del gap antes de revertir. Este movimiento inicial desde prev_close con volumen bajo es la senal de entrada.
TR75 = 0.75 × prev_range
1P = 0.25 × TR75
Entry SHORT = prev_close + 1P
03
Target — 75% Reversal
Si el precio mueve 25% en la direccion del gap con volumen bajo, estadisticamente completa el 75% hacia el extremo opuesto. Target = PIVOT ± TR/4.
Target SHORT = PIVOT − TR/4
Target LONG = PIVOT + TR/4
Vol filtro: ratio ≤ 1.0
