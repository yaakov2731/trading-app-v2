## What the app does

ES (E-Mini S&P 500 futures) daily trading plan calculator. User enters the previous day's High/Low/Close + today's Open (optional) and the app computes all levels for the 1P + PIVOT-TR system.

## Core calculation logic (in index.html `<script>`)

```
prev_range = H - L
gap        = O - C  (0 if no Open entered)
TR         = prev_range + |gap|
PIVOT      = H - TR/2
S1         = PIVOT - TR/4   ← SHORT target
R1         = PIVOT + TR/4   ← LONG target
S2         = PIVOT - TR/2
TR75       = 0.75 * prev_range
1P         = 0.25 * TR75    (= 0.1875 * prev_range)

entry_SHORT = C + 1P        stop_SHORT = H + TR*0.10
entry_LONG  = C - 1P        stop_LONG  = S2 - TR*0.10
```

Gap direction controls active setups: gap UP → SHORT only, gap DOWN → LONG only, flat → both.
